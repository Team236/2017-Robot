# Controls

## Left Joystick
| Button | Function |
|--------|----------|
| Trigger | Align Drive |
| Shift up/down | POV up/down |

## Right Joystick
| Button | Function |
|--------|----------|
| Trigger | Reverso Drive |
| Middle | Intake   |

## Controller
| Button | Function |
|--------|----------|
| LB | Spin up shooter |
| START | Climb |
| RT | Feed shooter | 
| Back | Toggle camera position |
| POV up/down | Garage Door up/down |
| POV left/right | Change shooter presets |
| Y | Retract intake |
| A | Deploy intake|
| B | Grasp gear |
| X | Release gear |
| Left Y | Adjust camera angle |
