# Controls

## Left Joystick
| Button | Function |
|--------|----------|
| Middle | Align Drive |
| POV up/down | Shift up/down |

## Right Joystick
| Button | Function |
|--------|----------|
| Trigger | Reverso Drive |
| Middle | Intake |

## Controller
| Button | Function |
|--------|----------|
| LB | Spin up shooter |
| START | Climb |
| RT | Feed shooter | 
| Back | Cycle camera position |
| POV up/down | Garage Door up/down |
| POV left/right | Change shooter presets |
| Y | Retract intake |
| A | Deploy intake|
| B | Grasp gear |
| X | Release gear |
